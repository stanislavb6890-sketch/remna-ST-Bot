# STEALTHNET Admin (Frontend)

Панель администратора и клиентский кабинет STEALTHNET 3.0.

## Стек

- **React 18** + **Vite**
- **Tailwind CSS v4**
- **[shadcn/ui](https://ui.shadcn.com/)** — UI-компоненты (Radix UI + стили)

Компоненты shadcn находятся в `src/components/ui/`. Конфигурация CLI — `components.json` в корне (добавление компонентов: `npx shadcn@latest add <компонент>`).

## Запуск

```bash
npm install
npm run dev    # разработка (порт 5173)
npm run build  # сборка в dist/
```
